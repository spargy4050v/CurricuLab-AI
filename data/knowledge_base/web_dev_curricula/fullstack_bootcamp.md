# Full-Stack Web Development Bootcamp - 6 Month Intensive

**Level:** Certification  
**Duration:** 6 Months (3 Semesters/Modules)  
**Total Credits:** 36

## Overview
An intensive bootcamp covering modern full-stack web development from fundamentals to deployment. Focuses on JavaScript ecosystem, React, Node.js, and cloud deployment.

## Module 1: Frontend Fundamentals (12 Credits - 8 Weeks)

### Week 1-2: Web Basics
- **WEB101** - HTML5 & CSS3 (2 credits) - Core
  - Semantic HTML, Flexbox, Grid, responsive design
  
- **WEB102** - JavaScript Fundamentals (3 credits) - Core
  - ES6+, DOM manipulation, async programming

### Week 3-4: Modern Frontend
- **WEB103** - React Fundamentals (3 credits) - Core - Prerequisites: WEB102
  - Components, hooks, state management, routing
  
- **WEB104** - CSS Frameworks (2 credits) - Core
  - Tailwind CSS, styled-components

### Week 5-8: Advanced Frontend
- **WEB105** - Advanced React (2 credits) - Core - Prerequisites: WEB103
  - Context API, Redux, performance optimization
  
- **WEB106** - Frontend Project (2 credits) - Lab - Prerequisites: WEB103
  - Build complete responsive web application

## Module 2: Backend Development (12 Credits - 8 Weeks)

### Week 9-10: Backend Basics
- **WEB201** - Node.js & Express (3 credits) - Core - Prerequisites: WEB102
  - REST APIs, middleware, authentication
  
- **WEB202** - Database Fundamentals (3 credits) - Core
  - SQL (PostgreSQL), NoSQL (MongoDB)

### Week 11-12: Advanced Backend
- **WEB203** - API Design (2 credits) - Core - Prerequisites: WEB201
  - RESTful design, GraphQL, API security
  
- **WEB204** - Authentication & Authorization (2 credits) - Core - Prerequisites: WEB201
  - JWT, OAuth, session management

### Week 13-16: Full Integration
- **WEB205** - Full-Stack Integration (2 credits) - Core - Prerequisites: WEB103, WEB201
  - Connect frontend to backend, state management
  
- **WEB206** - Backend Project (2 credits) - Lab - Prerequisites: WEB201, WEB202
  - Build complete API with database

## Module 3: Deployment & Advanced Topics (12 Credits - 8 Weeks)

### Week 17-18: DevOps Basics
- **WEB301** - Git & Version Control (1 credit) - Core
  - Git workflows, collaboration, branching strategies
  
- **WEB302** - Docker & Containers (2 credits) - Core
  - Containerization, Docker Compose

### Week 19-20: Cloud Deployment
- **WEB303** - Cloud Platforms (2 credits) - Core - Prerequisites: WEB205
  - AWS/Vercel/Netlify deployment, CI/CD
  
- **WEB304** - Testing (2 credits) - Core
  - Unit testing, integration testing, E2E testing

### Week 21-24: Capstone Project
- **WEB305** - Capstone Full-Stack Project (5 credits) - Lab - Prerequisites: All previous courses
  - Complete production-ready application
  - Team collaboration, agile methodology
  - Deployment to cloud platform

## Learning Outcomes
- Build responsive, modern web applications
- Develop RESTful APIs and microservices
- Work with both SQL and NoSQL databases
- Deploy applications to cloud platforms
- Follow industry best practices and coding standards
- Work effectively in development teams

## Career Paths
- Full-Stack Developer
- Frontend Developer
- Backend Developer
- Web Application Developer
- JavaScript Developer
- DevOps Engineer (Junior)

## Technologies Covered
- **Frontend:** HTML5, CSS3, JavaScript (ES6+), React, Redux, Tailwind CSS
- **Backend:** Node.js, Express, REST APIs, GraphQL
- **Databases:** PostgreSQL, MongoDB
- **DevOps:** Git, Docker, AWS/Vercel, CI/CD
- **Testing:** Jest, React Testing Library, Cypress
